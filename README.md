# Unraid Templates

Community Apps templates for Unraid.

## Available Templates

| App | Description |
|-----|-------------|
| [SeerrDiscordBot](https://github.com/pejkaa/seerr-discord-bot) | Discord bot for requesting media via Seerr |

## Installation

Add all templates at once:
```bash
wget -O /boot/config/plugins/dockerMan/templates-user/SeerrDiscordBot.xml \
  https://raw.githubusercontent.com/pejkaa/unraid-templates/main/SeerrDiscordBot.xml
```

Or install individually from the Unraid Community Apps store.
